What Are Sensors?

Sensor: A device that converts physical phenomena into measurable signals

Why robots need sensors:

  • Perception: Understand the environment
  • Navigation: Know where you are and where to go
  • Interaction: Respond to changes
  • Safety: Detect obstacles and hazards

Sensor Categories

Proprioceptive

Sense internal state

  • Wheel encoders
  • Motor current
  • Battery voltage
  • IMU (orientation)

Exteroceptive

Sense external environment

  • Distance sensors
  • Cameras
  • Temperature
  • Light sensors
Both types are essential for autonomous behavior!

Arduino Code: HC-SR04

#define TRIG_PIN 9
#define ECHO_PIN 10

void setup() {
  Serial.begin(9600);
  pinMode(TRIG_PIN, OUTPUT);
  pinMode(ECHO_PIN, INPUT);
}

void loop() {
  // Send trigger pulse
  digitalWrite(TRIG_PIN, LOW);
  delayMicroseconds(2);
  digitalWrite(TRIG_PIN, HIGH);
  delayMicroseconds(10);
  digitalWrite(TRIG_PIN, LOW);
  
  // Measure echo duration
  long duration = pulseIn(ECHO_PIN, HIGH);
  
  // Calculate distance in cm
  float distance = duration * 0.034 / 2;
  
  Serial.print("Distance: ");
  Serial.print(distance);
  Serial.println(" cm");
  
  delay(100);
}

Infrared (IR) Sensors

Principle: Emit IR light and measure reflection

Two main types:

Digital (Binary)

  • Obstacle: YES/NO
  • Adjustable threshold
  • Simple, fast

Analog (Distance)

  • Continuous distance
  • Voltage varies with distance
  • Typical range: 10-80 cm

IR Digital Sensor Wiring

IR Sensor Pin Arduino Pin
VCC 5V
OUT Any digital pin (e.g., 2)
GND GND 
void setup() {
  Serial.begin(9600);
  pinMode(2, INPUT);
}

void loop() {
  int obstacle = digitalRead(2);
  if (obstacle == LOW) {
    Serial.println("Obstacle detected!");
  }
  delay(100);
}

Soil Moisture Sensors

Principle: Measure soil conductivity (wetness)

Two types:

Resistive

  • Two metal probes
  • Cheap ($1-2)
  • Corrodes over time

Capacitive

  • No exposed metal
  • More durable
  • Slightly expensive
Output: Analog voltage (higher = drier soil)

Soil Moisture Sensor Code

void loop() {
  int moisture = analogRead(A0);
  
  Serial.print("Moisture: ");
  Serial.print(moisture);
  
  // Calibrate these thresholds for your sensor!
  if (moisture < 300) {
    Serial.println(" - Very Wet");
  } else if (moisture < 500) {
    Serial.println(" - Wet");
  } else if (moisture < 700) {
    Serial.println(" - Moist");
  } else if (moisture < 900) {
    Serial.println(" - Dry");
  } else {
    Serial.println(" - Very Dry");
  }
  
  delay(1000);
}
Tip: Calibrate by testing in water vs air

Multi-Sensor Systems

Real robots use multiple sensors together:

  • Redundancy: Backup if one fails
  • Complementary: Different strengths
  • Sensor Fusion: Combine for better accuracy
Example: Self-driving cars use cameras, lidar, radar, GPS, and IMU together

  1. Sensors are essential for robot perception
  2. Distance sensors: Ultrasonic (long range) vs IR (fast, short)
  3. Environmental sensors: Temperature, light, moisture
  4. Analog sensors output voltage proportional to measurement
  5. Digital sensors output binary (on/off) signals
  6. Calibration is crucial for accurate readings
  7. Multiple sensors provide redundancy and better perception
