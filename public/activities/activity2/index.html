Activity 2: Distance and IR Sensors

Overview

In this activity, you will wire and experiment with two types of distance sensors: an ultrasonic sensor (HC-SR04) and an infrared (IR) obstacle detection sensor. You’ll compare their performance with different objects, surfaces, and distances. This activity helps you understand how robots perceive their environment and the trade-offs between different sensor types.

Teams: Work in pairs (2 students per team), but each team must test both sensors.

Due Date: February 4th by 11:50 AM

Learning Objectives

By completing this activity, you will:

  • Wire and operate ultrasonic and IR distance sensors
  • Understand how different sensors detect obstacles
  • Compare sensor performance across various conditions
  • Document sensor capabilities and limitations
  • Learn when to use each sensor type in robotics applications

Requirements

What You Must Demonstrate & Submit

  1. Wire both sensors correctly to Arduino Uno
  2. Test ultrasonic sensor with different objects and distances
  3. Test IR sensor with different surfaces and obstacles
  4. Complete and submit the comparison table with your observations

Part 1: Ultrasonic Sensor (HC-SR04)

Understanding Ultrasonic Sensors

The HC-SR04 uses sound waves (40 kHz ultrasonic) to measure distance:

  1. Sends out a sound pulse
  2. Measures time for echo to return
  3. Calculates distance: distance = (time √ó speed_of_sound) / 2

Specifications:

  • Range: 2 cm to 400 cm
  • Accuracy: ¬±3 mm
  • Beam angle: ~15¬∞

Wiring the HC-SR04

Components:

  • Arduino Uno
  • HC-SR04 ultrasonic sensor
  • 4 jumper wires
  • Breadboard (optional)

Wiring Table:

HC-SR04 Pin Arduino Pin Purpose
VCC 5V Power
TRIG Any digital pin (e.g., Pin 9) Trigger pulse
ECHO Any digital pin (e.g., Pin 10) Echo response
GND GND Ground

Arduino Code for HC-SR04

// HC-SR04 Ultrasonic Sensor Code
#define TRIG_PIN 9
#define ECHO_PIN 10

void setup() {
  Serial.begin(9600);
  pinMode(TRIG_PIN, OUTPUT);
  pinMode(ECHO_PIN, INPUT);
  Serial.println("HC-SR04 Ultrasonic Sensor Test");
  Serial.println("Distance in cm:");
}

void loop() {
  // Send 10¬µs pulse to trigger
  digitalWrite(TRIG_PIN, LOW);
  delayMicroseconds(2);
  digitalWrite(TRIG_PIN, HIGH);
  delayMicroseconds(10);
  digitalWrite(TRIG_PIN, LOW);
  
  // Measure echo duration (in microseconds)
  long duration = pulseIn(ECHO_PIN, HIGH);
  
  // Calculate distance in cm
  // Speed of sound: 343 m/s = 0.0343 cm/¬µs
  float distance = duration * 0.034 / 2;
  
  // Display result
  Serial.print("Distance: ");
  Serial.print(distance);
  Serial.println(" cm");
  
  delay(500);  // Wait 500ms between readings
}

Upload this code and open the Serial Monitor (9600 baud) to see distance readings.

Testing the Ultrasonic Sensor

Experiment with:

  1. Different distances: 5 cm, 20 cm, 50 cm, 100 cm, 200 cm
  2. Different surfaces:
    • Flat wall
    • Angled surface (45¬∞)
    • Soft material (cloth, foam)
    • Shiny/reflective surface
  3. Different object sizes:
    • Large box
    • Thin rod or pencil
    • Your hand

Record your observations in the comparison table (see Part 3).

Part 2: Infrared (IR) Obstacle Sensor

Understanding IR Sensors

Digital IR sensors emit infrared light and detect reflections:

  • Output: HIGH (no obstacle) or LOW (obstacle detected)
  • Has a potentiometer to adjust detection threshold
  • Fast response time (~20 ms)
  • Typical range: 2-30 cm (adjustable)

Wiring the IR Sensor

Components:

  • Arduino Uno
  • IR obstacle detection sensor
  • 3 jumper wires

Wiring Table:

IR Sensor Pin Arduino Pin Purpose
VCC 5V Power
OUT Any digital pin (e.g., Pin 2) Detection signal
GND GND Ground

Arduino Code for IR Sensor

// Digital IR Obstacle Sensor Code
#define IR_PIN 2

void setup() {
  Serial.begin(9600);
  pinMode(IR_PIN, INPUT);
  Serial.println("IR Obstacle Sensor Test");
  Serial.println("Checking for obstacles...");
}

void loop() {
  int sensorState = digitalRead(IR_PIN);
  
  if (sensorState == LOW) {
    Serial.println("üö® OBSTACLE DETECTED!");
  } else {
    Serial.println("‚úì Clear (no obstacle)");
  }
  
  delay(300);
}

Upload this code and open the Serial Monitor (9600 baud).

Adjusting the IR Sensor Threshold

Most IR sensors have a small potentiometer (screw or knob):

  • Clockwise: Increases detection distance
  • Counter-clockwise: Decreases detection distance

To calibrate:

  1. Place an object at your desired detection distance
  2. Adjust the potentiometer until the sensor just detects it
  3. Test with other objects to verify consistency

Testing the IR Sensor

Experiment with:

  1. Different distances: Start close, move object away slowly
  2. Different colors:
    • White paper
    • Black paper
    • Colored objects
  3. Different materials:
    • Matte surfaces
    • Glossy/reflective surfaces
    • Transparent objects (water bottle)
  4. Detection angle: Move object side-to-side

Record your observations in the comparison table.

Tips for Success

  • Test in a quiet area: Background noise can affect ultrasonic sensors
  • Use Serial Monitor: Always check that your sensors are giving readings
  • Be systematic: Test one variable at a time
  • Document as you go: Don’t wait until the end to fill in your table
  • Calibrate the IR sensor: Adjust the potentiometer for best performance
  • Watch for interference: Keep sensors apart if testing both simultaneously

Troubleshooting

Ultrasonic Sensor Issues

Problem Possible Solution
Always reads 0 or 400+ cm Check wiring; verify TRIG and ECHO pins
Inconsistent readings Ensure stable power; test in quiet environment
Doesn’t detect certain objects Object may be too small or angled; try different angle

IR Sensor Issues

Problem Possible Solution
Always detects obstacle Adjust potentiometer counter-clockwise
Never detects obstacle Adjust potentiometer clockwise; check wiring
Inconsistent detection Calibrate for specific distance; check ambient light
